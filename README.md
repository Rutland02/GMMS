# 健身房会员管理系统 (GMMS 2.0)

一个基于Qt框架开发的健身房会员管理系统，提供会员管理、课程预约、签到统计等功能。

## 功能特点

### 核心功能
- **会员管理**：添加、查询、编辑和删除会员信息，支持会员等级与积分系统
- **课程管理**：管理健身课程信息
- **预约系统**：会员预约课程功能，预约成功可获得积分奖励
- **签到功能**：会员签到记录，签到成功可获得积分奖励
- **统计分析**：会员活跃度、课程预约情况、积分统计等数据统计
- **积分系统**：实现积分累积、等级升级机制，支持手动积分调整

### 界面模块
- **会员标签页**：管理会员基本信息，显示和调整会员积分与等级
- **课程标签页**：管理课程信息
- **预约标签页**：处理课程预约，预约成功自动积分
- **查询标签页**：查询会员和课程信息
- **统计标签页**：数据可视化和统计分析，包括积分统计

## 技术栈

- **开发框架**：Qt 5.14.2
- **开发语言**：C++
- **UI组件**：Qt Widgets
- **图表组件**：Qt Charts
- **数据存储**：JSON文件
- **构建工具**：MinGW 7.3.0

## 构建说明

### 环境要求
- Qt 5.14.2 或更高版本
- MinGW 7.3.0 编译器
- Windows 操作系统

### 构建步骤

1. **打开项目**
   使用Qt Creator打开`GymManager.pro`文件

2. **配置构建套件**
   选择MinGW 7.3.0构建套件

3. **构建项目**
   ```bash
   # 清理项目
   D:/QT/Tools/mingw730_64/bin/mingw32-make clean
   
   # 构建项目
   D:/QT/Tools/mingw730_64/bin/mingw32-make
   ```

4. **运行程序**
   构建完成后，可执行文件位于`release/GymManager.exe`

## 使用说明

### 会员管理
1. 进入会员标签页
2. 点击"添加会员"按钮添加新会员
3. 填写会员信息并保存

### 课程预约
1. 进入预约标签页
2. 选择会员和课程
3. 点击"预约"按钮完成预约

### 签到功能
1. 进入签到界面
2. 选择会员和课程
3. 点击"签到"按钮记录签到

### 数据统计
1. 进入统计标签页
2. 查看会员统计、课程统计和活跃度图表
3. 可选择日期范围查看特定时间段的数据

## 开发说明

### 数据持久化
系统使用JSON文件存储数据，在程序启动时加载数据，退出时保存数据。

### 类结构
- **Member**：会员类，存储会员信息
- **Course**：课程类，存储课程信息
- **CheckIn**：签到类，存储签到记录
- **GymData**：核心数据管理类，处理数据的加载、保存和统计

### 界面设计
使用Qt Widgets和布局管理器创建响应式界面，支持多标签页操作。

## 问题与解决方案

### 常见问题
- **编译错误**：确保使用正确的Qt版本和编译器
- **链接错误**：检查是否正确包含Qt Charts模块
- **数据丢失**：确保程序正常退出，数据会自动保存

### 开发建议
- 扩展功能时保持代码结构清晰
- 遵循Qt的编程规范
- 添加适当的错误处理机制
